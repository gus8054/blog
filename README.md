# 목적
프론트엔드와 백엔드를 분리시켜 개발하고 api로 네트워크 통신을 하는 프로젝트를 만들어 보자 시작한 프로젝트.

# 사용한 기술
## 프론트 엔드
* react
* react router
* react quill
* axios

## 백 엔드
* node.js
* express
* bcrypt
* jsonwebtoken

## 데이터베이스
* mysql
